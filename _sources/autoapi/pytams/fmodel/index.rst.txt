.. py:class:: ForwardModel(params: dict = None, ioprefix: str = None)


   A template class for the stochastic forward model.

   Implement the core methods required of the forward
   model within the TAMS context. Exception are thrown
   if those functions are not overritten in actual model.

   .. py:method:: advance(dt: float, forcingAmpl: float) -> float

      Advance function of the model.

      :param dt: the time step size over which to advance
      :param forcingAmpl: stochastic multiplicator

      :returns: Some model will not do exactly dt (e.g. sub-stepping) return the actual dt
